How to Generate a Grafana API Key

1

Login to Grafana

Open Grafana at http://localhost:3000 and login with your credentials (default: admin/admin)

2

Navigate to API Keys

Click on the gear icon (‚öôÔ∏è) ‚Üí Configuration ‚Üí API Keys

3

Create New API Key

Click "Add API key" button

4

Configure Key Settings

  • Key name: ITOM Integration
  • Role: Admin (required for full integration)
  • Time to live: Choose based on your security policy
5

Copy the API Key

Copy the generated API key immediately (it won't be shown again!) and paste it in the settings above
